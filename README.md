### 🚀 Featured Research & Projects

#### 1. Optimization of Continual Learning Strategy (ViT-CMS)
> Architected a novel Replay-Free Nested Learning system using a ViT-Base backbone. Integrated Continuum Memory Systems to mitigate catastrophic forgetting without storing raw data, achieving **73.1% F1-score** and **75.0% Stability** on CIFAR-10.

#### 2. Agentic RAG Framework for Legal Document Retrieval
> Developed a scalable QA system using **LangGraph, FAISS, and Llama-3.1** to autonomously extract facts from 20,000+ unstructured OCR-degraded documents. Deployed via a JWT-secured **Django REST API** on Hugging Face Spaces.

#### 3. Autonomous Driving System (RL & Vision Integration)
> Engineered a navigation system integrating **Deep Deterministic Policy Gradient (DDPG)** for continuous vehicle control with **YOLOv11** for real-time perception. Implemented robust PyTorch training pipelines with advanced augmentation.

#### 4. Vietnam AQI Forecasting System
> Architected a **Dual Embedding BiLSTM with Attention** to model complex spatio-temporal dependencies. Achieved **RMSE ~0.43** for a 7-day recursive forecast using a custom Weighted MSE/Huber Loss to handle outliers.
